# Re-Ranking Engine

## Purpose
Optimize recommendation lists beyond model scores
using business, fairness, and UX constraints.

## Components
- Diversity control
- Freshness bias
- Fairness & exposure balancing
- Hard business rules

## Why Re-Ranking?
Pure relevance maximization leads to:
- Content saturation
- Bias amplification
- Poor long-term engagement

## Ownership
This layer is owned by senior ML engineers
and product stakeholders.
